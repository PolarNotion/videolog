What is this?

Videolog is a jquery plugin that lets you view video activity from users on either HTML5 videos, Youtube, and/or Vimeo videos.

Installation

Download the production version or the development version.

  • HTML
  • JS
  • CSS 

//Select any iframe or video element by giving it a class name to use for the plugin to track.
//Use a unique id to idenity which video if you have multiple videos on the page.

<iframe id="vplayer" class="trackable-video" src="https://player.vimeo.com/video/76979871"></iframe>

<iframe id="ytplayer" class="trackable-video" src="https://www.youtube.com/embed/M7lc1UVf-VE?enablejsapi=1"></iframe>

<video id="video1" class="trackable-video" height="250" width="350" controls>
<source src="videos/video-example.mp4" type="video/mp4"> Your browser does not support HTML5 video. </video>

<script src="jquery.js"></script> // jquery
<script src="dist/videolog.min.js"></script> // video plugin
<script src="https://player.vimeo.com/api/player.js"></script> //vimeo api*

Youtube API is loaded asychronously with the plugin while Vimeo API script should be applied where videos are.

  $('trackable-video').videoLog();

Demo

Three different videos; Vimeo, Youtube, and HTML5 videos.

Videos are tracked by their class name but can be given a different class name if desired. The id of the element also does not matter but should be unique to to identify which video is being tracked to elminate confusion.
